# Jet

Jet is a tensor library for Crystal powered by CUDA and CuDNN.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  cudnn:
    gitihub: sherjilozair/jet
```

## Usage

```crystal
require "jet"
```
