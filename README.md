# cudnn

[WIP] Crystal bindings for NVIDIA CuDNN.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  cudnn:
    github: sherjilozair/cudnn.cr
```

## Usage

```crystal
require "cudnn"
```
